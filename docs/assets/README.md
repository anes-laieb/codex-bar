# README media

This directory contains presentation assets used by the project README.

- `codex-bar-hero.png` is an AI-generated product illustration based on the repository's
  supplied application artwork. It is decorative and is not presented as a screenshot.
- `app-window.png` is based on a screenshot of a locally built Codex Bar 2.1.8 window.
  Live task titles were replaced with demo-safe copy before publication; the application
  structure, controls, states, and preferences shown are representative of the real UI.

The application artwork under `app/StatusAssets/` remains the source of truth for icons
displayed by the application itself.
