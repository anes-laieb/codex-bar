# Codex Bar 1.0.0

Codex Bar 1.0.0 is the first stable public release: a native macOS menu-bar companion
that keeps every active Codex task visible, opens the exact task in one click, and tells
you when Codex is working or waiting for you.

## Highlights

- **Every task in one place:** follow multiple open Codex tasks with live state, elapsed
  time, model, effort, project, and session information.
- **Exact-task navigation:** open an existing conversation or start a new chat through
  native `codex://` routes.
- **Useful attention controls:** completion and question notifications, separate sounds,
  notification actions, per-session mute controls, and quiet hours.
- **Personal organization:** pin, hide, restore, and review recent task activity without
  leaving the menu bar.
- **Three icon appearances:** Default, adaptive System, and animated Sky Blue artwork.
- **Responsive sessions window:** long session titles and metadata truncate cleanly at
  narrow widths without wrapping into adjacent rows.

## Verified in-app updates

Clicking **Update Available** downloads and installs the matching release directly. The
app shows download and verification progress, supports cancellation before installation,
and relaunches automatically when replacement succeeds.

Before changing the installed application, Codex Bar verifies:

- the architecture-specific GitHub release filename and asset size;
- GitHub's SHA-256 asset digest;
- the extracted bundle identifier and release version;
- the executable architecture and complete code signature.

The existing application remains available for rollback until the new version launches.
If the installation directory is not writable, Codex Bar leaves the app untouched and
offers the release page as a manual fallback.

## Download

| Asset | Platform | Architecture | Minimum OS |
| --- | --- | --- | --- |
| `Codex-Bar-1.0.0-macOS-arm64.dmg` | macOS installer | Apple silicon (`arm64`) | macOS 13 |
| `Codex-Bar-1.0.0-macOS-arm64.dmg.sha256` | DMG checksum | SHA-256 | — |
| `Codex-Bar-1.0.0-macOS-arm64.zip` | In-app update | Apple silicon (`arm64`) | macOS 13 |
| `Codex-Bar-1.0.0-macOS-arm64.zip.sha256` | ZIP checksum | SHA-256 | — |

For a standard installation, verify the DMG and drag **Codex Bar.app** onto its
**Applications** shortcut:

```sh
shasum -a 256 -c Codex-Bar-1.0.0-macOS-arm64.dmg.sha256
```

The build is ad-hoc signed and is **not Apple-notarized** because the project does not
currently have a Developer ID signing identity. On first launch, use
**Control-click → Open** if macOS asks you to confirm the application.

## Privacy and network behavior

Session discovery and status processing happen locally. Codex Bar reads Codex's thread
index and rollout logs without uploading their contents or modifying Codex configuration.
Automatic checks request public release metadata from GitHub; the ZIP is downloaded only
after the user clicks **Update Available**.

## Verification

The release is produced by `scripts/package-release.sh`, which performs a clean optimized
Swift build, validates `Info.plist`, verifies the app signature and architecture, checks
the bundled license, notice, and artwork, validates ZIP and DMG integrity, and emits
separate SHA-256 checksums.
