# Contributing to Codex Bar

Thank you for taking an interest in Codex Bar.

## Current contribution status

Codex Bar is **not accepting pull requests or other code/documentation contributions
right now**. The maintainers are currently keeping implementation work within the core
team so the application and its release process can stabilize.

Please do not open a pull request, including a draft pull request. Unsolicited pull
requests may be closed without review. This policy may change in the future and this
file will be updated when it does.

The Apache-2.0 license still permits you to fork and modify the project for your own
use, subject to the license terms. The policy above only describes what the upstream
project is currently able to review and merge.

## Issues are welcome

You are encouraged to open an issue for:

- reproducible bugs;
- feature requests and product feedback;
- documentation gaps or incorrect instructions;
- compatibility problems with a Codex or macOS release.

Before opening an issue, search the existing issues to avoid duplicates. Include:

- your macOS version and Mac architecture;
- whether you use Codex Desktop, Codex CLI, or both;
- the Codex Bar and Codex versions;
- exact reproduction steps, expected behavior, and actual behavior;
- relevant screenshots or sanitized logs.

Codex rollout logs can contain prompts, paths, model responses, and other private data.
Remove secrets and personal information before attaching any excerpt. Do not report a
security vulnerability in a public issue; follow [SECURITY.md](SECURITY.md) instead.

Participation in the issue tracker is subject to [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
