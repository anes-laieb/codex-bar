# Community Code of Conduct

## Our standard

Codex Bar's issue tracker should be a welcoming, practical place to report problems
and discuss ideas. Be respectful, patient, and specific. Assume good intent, focus on
the work, and make room for people with different backgrounds and levels of experience.

Unacceptable behavior includes harassment, discrimination, threats, personal attacks,
sexualized language or imagery, deliberate intimidation, doxxing, and publishing
someone else's private information without permission.

## Scope and enforcement

This code applies to all Codex Bar project spaces and to public behavior when someone
is representing the project. Maintainers may edit, hide, or remove content and may
temporarily or permanently restrict participation when conduct is inappropriate.

For ordinary moderation concerns, flag the relevant GitHub content for the maintainers.
If a report involves sensitive security information, use the private process in
[SECURITY.md](SECURITY.md). Reports will be reviewed as promptly and fairly as the
maintainers can manage. Retaliation against someone who makes a good-faith report is
not permitted.
