# Support

Codex Bar is a community project and does not provide guaranteed or private end-user
support.

For a reproducible bug, compatibility problem, documentation error, or feature request,
search the [GitHub issues](https://github.com/anes-laieb/codex-bar/issues) and open a new
issue if one does not already exist. Include the details requested in
[CONTRIBUTING.md](CONTRIBUTING.md).

For installation or runtime problems, first confirm that:

1. the Mac is running macOS 13 or newer;
2. `swiftc`, `sqlite3`, and `lsof` are available;
3. Codex Desktop or Codex CLI has at least one active task;
4. notification permission is enabled for Codex Bar in System Settings;
5. rebuilding with `./install-app.sh` reproduces the problem.

Do not use a public issue for vulnerabilities. Report those privately as described in
[SECURITY.md](SECURITY.md).

The project currently accepts issues but does not accept pull requests. See
[CONTRIBUTING.md](CONTRIBUTING.md) for the current policy.
